Here are some useful links I use quite frequently

General Research

  • arXiv pre-print server - for newly submitted astrophysics papers
  • Astrophysics Data System - search for all astrophysics papers
  • LaTeX Tutorial - if you don't know LaTeX, learn it now!
  • Astrobites - arXiv reader's digest, written by graduate students for undergrads
  • Mendeley - organizer for papers and references

Coding

  • Software Carpentry - various coding tutorials
  • Practical Python for Astronomers - an excellent, applications-first tutorial
  • Python Tutorial - developed by python.org
  • Linux Command Directory - index of useful Linux terminal commands
  • LinuxCommand.org - A useful tutorial on using Linux command-line terminal
  • Code Academy - tutorials on Python, HTML, etc
  • Coursera - extensive online courses in computer science and statistics available
  • PyFITS Documentation - Python FITS analysis package
  • Matplotlib-Pyplot Documentation - API for python plotting module
  • Pyplot Mathtext - LaTeX-like symbols for Pyplot
  • Pyplot Errorbars - a good example with source
  • PyEphem Documentation - API for python ephemerides calculator

Math/Physics Resources

  • Haverford Math Appendices - guides to mathematical tools for physics
  • Ohio State GRE Prep - GRE Physics practice questions
  • Conquering the Physics GRE - the ONLY good GRE Physics review book
  • Physics Teaching Aids - practical hands-on activities
  • Feynman Lectures - collection of Feynman resources, including lab notes and exams

Graduate School Search Resources

  • How to choose an astronomy grad program - Advice from an astro professor for deciding on which program to attend, after being accepted
  • AIP Astronomy Roster Archive - Detailed astro program census data from last several decades: how many grad students and PhDs awarded at each school per year
